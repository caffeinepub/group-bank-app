# Specification

## Summary
**Goal:** Build a simplified group banking app for managing up to 26 members with monthly installment tracking, bilingual support (Hindi/English), and admin controls.

**Planned changes:**
- Create member management system storing name, mobile number, joining date, and profile photo for up to 26 members
- Implement monthly installment tracking (₹1000 per member) with payment history and total collection calculations
- Build individual member detail pages showing profile, payment history, and balance information
- Add login system using mobile number as username and last 4 digits as password (admin: 9315341037/1311)
- Create admin panel for member management, payment overview, and summary statistics
- Implement bilingual interface with Hindi/English language toggle
- Display UPI payment QR code (9315341037@ybl) on payment page
- Design clean, trustworthy visual theme with clear typography for financial data

**User-visible outcome:** Users can log in with their mobile number, view their payment history and member details, make payments via QR code, and toggle between Hindi and English. Admin can manage all members, add new members, and view comprehensive payment statistics.
